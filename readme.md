# 🏉 Rugby Performance Analytics Dashboard

Dashboard interativo para análise de desempenho de rugby utilizando dados da API Catapult OpenField.

## ✨ Funcionalidades

- 🔐 Autenticação via token JWT da Catapult
- 📊 Visualização de 500+ atletas e 1200+ atividades
- 🎯 Filtros em cascata: Temporada → Tipo → Atividade → Posição → Jogador
- 🏆 Métricas de eventos: Scrum, Contact, Kick, Lineout, RHIE
- 🏟️ Mapa interativo do campo de rugby com localização dos eventos
- 📋 Tabela detalhada com timestamps e métricas
